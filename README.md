# Maraline E-Commerce Platform

Maraline, ASP.NET Core MVC tabanlı geliştirilmiş, backend odaklı bir e-ticaret platformudur. 
Proje; kullanıcı yönetimi, ürün ve sipariş süreçleri, rol bazlı yetkilendirme ve admin paneli gibi temel e-ticaret fonksiyonlarını içermektedir.

## Features
- User registration & authentication
- Product & category management
- Shopping cart & order process
- Admin panel
- Role-based authorization

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- HTML, CSS, JavaScript

## Notes
Some core business logic (bonus & referral distribution) is intentionally excluded 
from this repository for security and business reasons.
